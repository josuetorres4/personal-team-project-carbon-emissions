# Gap Analysis

## Overview
Through prompt testing and peer discussions, we identified consistent weaknesses across existing AI tools when estimating carbon emissions.

---

## Major Gaps Identified

### 1. Lack of Transparency
Most tools provide emission totals without clearly showing formulas or emissions factors used. Users cannot verify results.

### 2. Poor Handling of Incomplete Data
When users provide partial information like cost instead of kWh, tools make assumptions silently rather than communicating uncertainty.

### 3. Inconsistent Results
Different tools produced significantly different emissions estimates for identical inputs, reducing user trust.

### 4. Limited Actionability
Tools provide general sustainability advice rather than personalized, impact-ranked recommendations.

### 5. Greenwashing Risk
Some tools assist in generating misleading sustainability claims without requiring verification.

---

## Impact on Users
Users reported difficulty trusting results and uncertainty about which behavioral changes actually reduce emissions.

---

## Key Insight
Existing tools answer questions but do not guide decision making or provide verifiable emissions reporting.

This creates an opportunity for a transparency first carbon emissions assistant.
