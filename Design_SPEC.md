# Design Specification — Carbon Emissions Assistant

## Concept
An AI-powered assistant that helps users estimate, understand, and reduce personal carbon emissions through transparent calculations and actionable insights.

---

## Target Users
- Students and commuters
- Environmentally conscious individuals
- Teams preparing sustainability reports

---

## User Journey

1. User opens application
2. Inputs electricity and commute data
3. System normalizes information
4. Emissions estimate generated
5. Assumptions displayed
6. Personalized reduction actions suggested
7. User tracks improvement over time

---

## Core Features
- Emissions calculator
- Transparency dashboard
- Action recommendation engine
- Progress tracking

---

## Key Screens
- Landing Page
- Input Wizard
- Results Dashboard
- Recommendations Page
- Progress Tracker

---

## Design Principles
Inspired by:
- Google Material Design
- IBM Carbon Design System
- Apple Human Interface Guidelines

Focus areas:
- clarity
- trust
- minimal cognitive load
